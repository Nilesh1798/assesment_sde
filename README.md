# React + Spring Boot

This File contain two folder Backend and Frontend 
for starting the project 
# Backend
1. cd backend
2. mvn clean install
3. mvn spring-boot:run

we have to run all 3 command one by one 

for frontend
# Frontend
1. cd frontend
2. npm install axios
3. npm start
